Templar's Fantastic Bows
Templar's Fantastic Bows is a mod currently still in development for the game Terraria. This mod will add nine new bows into the game, as well as one additional crafting component. The purpose of this mod is to add a bow crafting chain similar to the sword crafting chain that already exists. Future updates to the mod may include new accesssories and NPCs.

3D Connect Four
3D Connect Four is a game developed by myself and four others that aims to put a new twist on the classic game of Connect Four. My personal contribution to this project was placing pieces based on where the grid is clicked, generating the 3D grid, managing the 3D array, and adding the sliders that adjust what parts of the 3D grid are visible. (Note: Victory detection was never finished however everything else works as intended.)
