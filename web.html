Pokemon Card Finder
The Pokemon Card finder that I made is used to look at the any of the Pokemon cards that are currently in the competetive standard rotation. It utilizes the Pokemon TCG API (which can be found at docs.pokemontcg.io) to obtain data that is neatly displayed on the right based on what is entered on the left. The purpose of this project was to further learn how to use JavaScript in conjunction with HTML and CSS. This also helped me learn how to properly utilize local storage.
